# Admin Dashboard UI

A modern Admin Dashboard User Interface built using HTML5 and CSS3. This project was created as part of my frontend development learning journey and focuses on building a structured multi-page dashboard with clean layouts, navigation, forms, and management sections using only HTML and CSS.

## Features

* Login Page
* Dashboard Overview
* Sidebar Navigation
* Orders Management Page
* Products Management Page
* Customers Management Page
* Settings Panel
* Profile Settings Form
* Notification Settings
* Terms & Conditions Section
* Privacy Policy Section
* Clean and Organized User Interface
